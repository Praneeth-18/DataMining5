### Tabular Data (diverse) <a name="tabular-data"></a>

- **Description**: This dataset contains tabular data from NYC Taxi trips, including fare amounts, pickup and dropoff locations, trip distances, and more.
- **EDA**: Explored various columns, identified outliers, analyzed distributions of fares and trip distances.
- **Clustering**: Grouped trips based on similarities in trip patterns.
- **Modeling with AutoML**: Predicted fare amounts and trip durations.

---

### Time Series Data <a name="time-series-data"></a>

- **Description**: This dataset contains time-sequenced data points.
- **EDA**: Analyzed trends, seasonality, and anomalies in the data.
- **Clustering**: Identified patterns in different time series sequences.
- **Modeling with AutoML**: Forecasted future data points.

### Spatio-Temporal Data <a name="spatio-temporal-data"></a>

- **Description**: Data that has both spatial and temporal components.
